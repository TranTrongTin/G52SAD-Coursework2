In order to get access to all functions of the project, you need to add the "sqlite-jdbc-3.8.11.2.jar" 
file to the external library as well as run the IDE (Eclipse/Netbeans/IntelliJ) as administrator because 
I use this database to get some information in the project.

In PhotoViewController, line 395: you have to change the language level to level 8 to support the lambda expression.
